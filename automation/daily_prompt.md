# Smart Laser Daily 8AM Intelligence Brief Prompt

Generate the **Smart Laser Daily 8AM Brief** for AI data center power-infrastructure enclosures.

- Timezone context: **America/Chicago (Central Time)**.
- Coverage window: last 24 hours.
- Geographic priority: Texas/DFW first, then U.S.
- Must review:
  - Dallas Business Journal (signal until corroborated if paywalled)
  - Data Center Dynamics
- Source hierarchy:
  - Tier 1: standards bodies, AHJ code pages, OEM and investor primary docs
  - Tier 2: major trade outlets and analysts
  - Tier 3: social/job/forum signal only until corroborated

## Required output format (max ~2 pages)
1. Executive summary
2. What changed
3. Why it matters (technical / commercial / operational)
4. Recommendation (what Smart Laser should do this week)
5. Evidence (links, source tier, date)
6. Risks / unknowns
7. Next actions

## Include always
- Top 3 opportunities
- Top 3 risks
- Decisions needed
- Any "INTERNAL ASSUMPTION CONFLICT – REQUIRES MANAGEMENT DECISION."
- Any "CODE VERSION CONFLICT – VERIFY WITH AHJ BEFORE DESIGN FREEZE."
